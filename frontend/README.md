# Population Registry Frontend

This is the frontend for the Population Registry application, built with React and TypeScript.

## Setup and Running

1.  **Navigate to the frontend directory:**
    ```bash
    cd frontend
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the application:**
    ```bash
    npm start
    ```

The application will be running at `http://localhost:3000`.
